# 🚀 NASA API Project #

## 📌 Overview ##

This project uses NASA’s public APIs to collect and analyze space data, including the Astronomy Picture of the Day (APOD) and Near Earth Objects (Asteroids).

### 🌌 APIs Used ###
APOD (Astronomy Picture of the Day)
NeoWs (Asteroids Data)
🔗 https://api.nasa.gov/
### 🎯 What This Project Does ###
Fetches data using NASA APIs
Displays APOD image in notebook
Extracts asteroid information
Stores data in a pandas DataFrame
Cleans and processes data
Exports results to CSV

Pandas
Jupyter / Google Colab
